# Learnings

This project reinforced several important cloud and engineering concepts:

- Designing serverless microservices
- Integrating frontend and backend systems
- Using pre-signed URLs for secure image uploads
- Structuring DynamoDB data models
- Deploying static sites on Vercel
- Engineering cost-optimized solutions
- Implementing graceful fallback systems

It also provided hands-on experience in debugging and operational thinking.
