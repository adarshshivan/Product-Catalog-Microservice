# Why This Project

This project demonstrates practical, production-grade cloud engineering concepts while staying cost-optimized for long-term showcasing.

## Key Reasons
- Serverless architectures are widely used in modern cloud systems.
- CRUD microservices form the backbone of real-world applications.
- Pre-signed URL workflows are essential for secure file handling.
- Demonstrates full-stack integration: Frontend + Backend + Storage.
- Demo Mode ensures the project remains functional indefinitely without incurring AWS costs.

This project improves my ability to design, deploy, optimize, and retire cloud infrastructure responsibly.
