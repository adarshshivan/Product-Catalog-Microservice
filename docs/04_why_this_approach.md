# Why This Approach

The selected serverless architecture aligns with cloud best practices and optimizes for cost, scalability, and operational simplicity.

## Key Reasons
- No need to manage servers or containers.
- DynamoDB provides high-speed NoSQL storage ideal for product catalogs.
- S3 supports secure file uploads using pre-signed URLs.
- Lambda ensures minimal operational overhead.
- API Gateway simplifies routing, security, and versioning.
- Perfect fit for portfolio projects that must remain cost-neutral.

This approach allowed for rapid development, clean separation of concerns, and long-term sustainability.
