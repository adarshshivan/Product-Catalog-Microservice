# Tech Stack

## Frontend
- HTML5
- CSS3
- Vanilla JavaScript
- Vercel (static deployment)

## Backend
- AWS Lambda (Python)
- Amazon API Gateway (REST API)
- Amazon DynamoDB (NoSQL storage)
- Amazon S3 (image storage)
- IAM Roles (least-privilege access control)

## Testing and Debugging
- Postman (API testing and request validation)

## Development Tools
- VS Code
- AWS Console
- GitHub for version control

This stack delivers an efficient combination of serverless scalability, rapid development, clean deployment workflows, and enterprise-grade API testing.
